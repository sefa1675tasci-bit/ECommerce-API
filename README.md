## 🇬🇧 English

This project is a simple E-Commerce API built with ASP.NET Core Web API.

### Features

#### Product Management

- Get all products
- Get product by id
- Create product
- Update product
- Delete product
- Search products

#### Category Management

- Get all categories
- Get category by id
- Create category
- Update category
- Delete category

### Technologies Used

- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- AutoMapper
- Swagger / OpenAPI
- Repository Pattern
- Service Layer Architecture

# ECommerce API

## 🇹🇷 Türkçe

Bu proje, ASP.NET Core Web API kullanılarak geliştirilmiş basit bir E-Ticaret API uygulamasıdır.

### Özellikler

- Ürün (Product) yönetimi
  - Ürün listeleme
  - Ürün detay görüntüleme
  - Ürün ekleme
  - Ürün güncelleme
  - Ürün silme
  - Ürün arama

- Kategori (Category) yönetimi
  - Kategori listeleme
  - Kategori detay görüntüleme
  - Kategori ekleme
  - Kategori güncelleme
  - Kategori silme

### Kullanılan Teknolojiler

- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- AutoMapper
- Swagger / OpenAPI
- Repository Pattern
- Service Layer Architecture
